ENGLISH:

This ocx control is designed to make skins in Visual Basic 6 forms in a very simple way.
Copy the file vbskfree.ocx into the Windows\System folder and vbskfree.chm into the Windows\Help folder, or place them in the folder that you want.
To include the control in a VB project go to the 'Project' menu, select 'Components', then press browse and select the file vbskfree.ocx. After that you will be able to add the control to the forms.


The files included in the package:

1) this file Readme.txt
2) vbskfree.ocx (program file)
3) vbskfree.chm (help file)
4) vbskfree.DEP (dependency file)

ESPAÑOL:

Este control ocx está diseñado para hacer skins en formularios de Visual Basic 6 de una manera muy sencilla.
Copie el archivo vbskfree.ocx en la carpeta Windows\System y el archivo vbskfree.chm en Windows\Help, o déjelos en la carpeta que prefiera.
Para incluir el control en un proyecto VB diríjase al menú 'Proyecto', seleccione 'Componentes', luego presione el botón 'Examinar', busque y selecccione vbskfree.ocx. Luego de eso estará en condiciones de agregar el control a los formularios.

Los archivos incluidos en este paquete son:

1) este archivo Readme.txt
2) vbskfree.ocx (archivo del programa)
3) vbskfree.chm (archivo de ayuda)
4) vbskfree.DEP (archivo de dependencias)
